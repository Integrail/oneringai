# Multimedia Studio & Agent Improvements

- **Multimedia Connector Selector** — Choose which connector to use for image, video, and TTS generation. No more guessing when you have multiple API keys for the same vendor.
- **Audio Streaming** — Basic audio streaming support for text-to-speech during agent conversations.
- **Voiceover Playback** — Agents can speak responses aloud with TTS. Toggle speaker icon in chat, skip mid-sentence, supports multiple formats (MP3, Opus, AAC, FLAC, WAV).
- **Tool Categories & Catalog** — Reorganized tool system with categories (filesystem, web, code, desktop, etc.) and dynamic tool catalog for agents.
- **Routine Tools Migration** — Routine execution tools updated to the new tool registration system.
- **Better Agent Error Handling** — Improved error messages and recovery when agents encounter issues during execution.
- **User Info Plugin** — New agent context plugin for persistent user preferences across sessions.
- **Chat History Page** — Browse and search past conversations.
