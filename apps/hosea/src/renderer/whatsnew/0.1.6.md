# Multi-Tab Agents & MCP Servers

- **Multi-Tab Chat** — Run multiple agent conversations in parallel, each with its own context, memory, and session. Switch between tabs instantly.
- **MCP Servers** — Connect to Model Context Protocol servers (stdio and HTTP). Manage, connect, and discover tools from MCP endpoints.
- **Universal Connectors** — New vendor catalog with 35+ services (Slack, GitHub, Stripe, Salesforce, etc.). Create connectors from templates with guided setup.
- **Multimedia Studio** — Generate images, video, and speech from a dedicated studio page. Supports OpenAI, Google, and Grok models.
